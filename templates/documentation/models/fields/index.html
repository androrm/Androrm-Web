Field types

Androrm comes with a variety of predefined field types, that you need to use, in order to get the cool database support. With these fields androrm knows, how to build the database and is able to create relations between your model classes. So, by default there are two different types of fields. The ones, that only cope with data and the ones, that define your relations.
