...just have a look in the side menu. There you find information about basic model functions, detailed descriptions of all field types and (maybe the best part) how to write queries. Spoiler alert! No SQL here.

What you might recognize is, that the whole androrm API resembles django. This indeed was intentional. I think django is just a great web-framework and even this site here was written using django. So, if you are familiar with their way of doing things, you won't have any problems, to use androrm for your android app. Enjoy!
