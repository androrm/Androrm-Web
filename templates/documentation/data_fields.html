This document describes the different types of data fields, that are available in androrm. You can quick jump to any field reference using the respective link in the side menu.

General Notice

All data fields come with two functions 
get()
and 
set( value )
that can be used to retrieve or set their value. As these functions are trivial, they won't appear in each reference for a field.
