Androrm installation

Adding androrm to your project is as simple as 1, 2, 3. The only descission, you have to make right from the start is, whether you want to go with the latest stable release or if you want to check out the latest version in the repository.

Now, after you have decided for one version, the steps are exactly the same. Assuming you work within eclipse to write your android application, first create a new folder called libs. Right-click your project and choose New > Folder.

After that, just copy and paste the androrm.jar file into this folder and add it to the build path. You simply do this, by right-clicking the jar and selecting Build Path > Add to Build Path.

That's it! You're done and can now start defining your models.
