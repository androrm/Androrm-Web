You think androrm is a cool tool and you also have lots of ideas on how to improve it? If yes, then this is your section! Go ahead and fork the project on github and maybe become part of the core development team.

Or did you find a defect or something you want to have improved? Then file a bug or a feature request and help finding the parts, we forgot to cover!

Please note, that we are using Phabricator for everything from issue tracking, till task management. It is an open source tool, that was build at facebook. As androrm, it is in a very early stage, but we are very confident. Plus: you can easily log in, using your facebook account!



We recently got some feedback, that the structure of phabricator is hard to understand. Well, they kind of refused to use the terms, that you would normally expect in a project management tool. So here's a quick start guidline.

Login

In order to file anything you have to log in. You can do so with your facebook account. Don't worry, we are not at all interested in you personal data or pictures. This is just for convenience. Plus, it is way easier to get back to you, if any questions concerning your reported bug come up.

Core Components

I think this is the part where most people get lost. The fancy naming of things. So to make things clear, most of the time you will want to use one of these features:
  • Differential - Submitting diffs of your code, that might fix a problem
  • Diffusion - Browse through the androrm repo and view version history
  • Maniphest - This is the task tracker. Any reports go in this section
  • Herald - A tool, that watches for certain patterns and "looks for danger"
  • Phriction - The wiki

Creating a Bug-Report

In order to create a bug-report go to Maniphest on the main page.

Step 1

You are presented with a list of Tasks, that are currently assigned to you. So by default, you shouldn't see that much. No just go straight ahead and click the neat green button, that says "Create Task".

Step 2

After that, please provide as much information as possible. Also make sure, that you assigned the task to Philipp and that you have chose androrm as the project. If you have a stack tracce, please insert it. If you have any images or other files, that could support your report, please do not hesitate to upload them.
