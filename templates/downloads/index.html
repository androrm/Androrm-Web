Get your copy of androrm

Androrm is an object relational mapper for android. But it does not require special android specific features. It has been tested under android versions 2.1 up to 4.0.

If you run into any truble, please report the error or file a bug.

Get the latest official release

The fastest way, to get your copy of androrm, is to execute the following commands in a shell, if you work on a unix based operation system like Linux or Mac OS. 

{% include "downloads/examples/download.sh" %}

{% include "downloads/examples/github.sh" %}
