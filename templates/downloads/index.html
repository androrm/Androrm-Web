Androrm is an object relational mapper for android. But it does not require special android specific features. It has been tested under android versions 2.1, 2.2 and 2.3, but should also run on earlier versions of the operating system.

If you run into any truble, please report the error or file a bug.

Get the latest official release

The fastest way, to get your copy of androrm, is to execute the following commands in a shell, if you work on a unix based operation system like Linux or Mac OS. 
    >> wget http://androrm.the-pixelpla.net{% url download_latest_tarball %}
    >> tar -xvzf androrm_{{ version }}.tar.gz
    >> cd androrm_{{ version }}
    >> mv androrm.jar /path/to/your/lib/folder
Otherwise you can also download the file and extract it manually.

Get the latest development version

Androrm is completely open source and is distributed under the MIT License. It is hosted in a public repository on github.com. So if you want to check out the latest development version of the code, open a shell and execute the following command: 
       >> git clone git://github.com/pxlplnt/androrm.git
